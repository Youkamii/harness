# harness — 개인 하네스 모노레포

**하네스(harness)** = AI 코딩 에이전트(Claude Code 등)를 감싸는 규칙·안전장치·작업절차 묶음. `에이전트 = 모델 + 하네스` — 같은 모델도 하네스가 좋으면 결과가 달라진다. 이 저장소는 내 모든 하네스를 한곳에서 관리한다.

| 디렉터리 | 상태 | 내용 |
|---|---|---|
| [`claude-harness-2026-07-11/`](claude-harness-2026-07-11/) | **현행** | Claude Code 자율형 하네스. 무엇이고 어떻게 작동하는지는 [상세 README](claude-harness-2026-07-11/README.md) 참고 — 헌법(행동 규칙), 훅(파괴 명령·비밀 커밋 자동 차단), 스킬(기능 개발 파이프라인), 서브에이전트(적대 리뷰·독립 검증) |
| [`codex-harness-2026-07-11/`](codex-harness-2026-07-11/) | 은퇴(보존) | 전신 codex 하네스 — 오케스트레이터·evals 등 재이식 후보 자산 보관용 |

## 빠른 시작

```powershell
# Windows (Node.js 필요) — 현행 하네스를 이 PC의 Claude Code에 설치
node claude-harness-2026-07-11\install.mjs
```

설치는 멱등(몇 번 돌려도 안전)이며 기존 `~/.claude`의 다른 설정은 건드리지 않는다. 새 세션부터 적용된다.

## 운영 규칙

- **이 레포가 소스, `~/.claude/`는 배포본.** 로컬 배포본만 고치면 다음 설치 때 사라진다 — 반드시 여기를 고치고 install로 배포.
- 가드(훅) 패턴 수정은 테스트 케이스 먼저 (`claude-harness-*/hooks/*.test.*`).
- 새 하네스 버전을 만들면 디렉터리를 추가하고 이 표의 "현행"을 옮긴다.
